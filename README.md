# Shopapi

Simple shop api for Shopify internship assignment.

## Docs
[here](docs/README.md)

## How to use
To build and run the project:
```
make dependencies
make generate 
make shopapi
./shopapi serve --address=0.0.0.0:8000
```
or to run the tests:
```
make test-dependencies
make test
```
